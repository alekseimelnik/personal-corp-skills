# Grill Me

Makes the agent interview you relentlessly about a plan or design until you reach shared understanding. It walks down each branch of the decision tree, resolving dependencies between decisions one by one, asks one question at a time, and always provides its recommended answer so you can just say "A".

If a question can be answered by exploring the codebase, the agent explores instead of asking.

Use it to stress-test a plan before building: every unresolved fork becomes an explicit decision with a rationale.
