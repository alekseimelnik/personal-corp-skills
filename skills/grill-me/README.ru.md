# Grill Me

Заставляет агента безжалостно интервьюировать тебя по плану или дизайну, пока не будет достигнуто общее понимание. Агент идёт по каждой ветке дерева решений, разруливая зависимости между решениями по одному, задаёт вопросы строго по одному и к каждому даёт свой рекомендованный ответ — чтобы можно было ответить просто «А».

Если на вопрос можно ответить, изучив кодовую базу, — агент изучает сам, а не спрашивает.

Используй, чтобы прожарить план до начала работы: каждая нерешённая развилка превращается в явное решение с обоснованием.
